### 交换排序

|      项目      |          冒泡排序          |           快速排序           |
| :------------: | :------------------------: | :--------------------------: |
|     稳定性     |            稳定            |          ==不稳定==          |

| 时间效率(最好) |            O(n)            |         O(n*log~2~n)         |

### 归并排序和基数排序
